# KIOSCO

E-commerce de moda en Bootstrap 5 — galería de looks comprables como un todo.

**Anclaje:** *"La revista de moda que sí puedes comprar."*

## Páginas

- `index.html` — portada: hero collage + carrusel + FAQ
- `secciones/productos.html` — catálogo de 16 looks con filtros
- `secciones/detalle.html` — anatomía del look (Cuero Urbano)

## Stack

- Bootstrap 5.3.3 (CDN)
- HTML5 semántico
- CSS propio único en `css/estilos.css`
- JS vanilla (`js/principal.js`, `js/tienda.js`) — sin servidor
- Tipografías locales en `tipografias/` (Archivo + Hanken Grotesk + Space Mono)

## Estructura

```
/
├── index.html
├── favicon.svg
├── proyecto.md           Descripción bilingüe del proyecto
├── secciones/            productos.html, detalle.html
├── css/estilos.css       Único archivo CSS
├── js/                   principal.js, tienda.js
├── multimedia/           Imágenes (3 tamaños cada una)
├── tipografias/          .woff2 locales
└── wireframes/           PDFs de baja fidelidad por dispositivo
```
